# 🏪 Sports Store (WPF)

Приложение для управления магазином спорттоваров.

## 🚀 Как запустить

1. Открой `SportsStoreApp.sln` в Visual Studio
2. Нажми **F5**
3. Готово!

**Ничего устанавливать не нужно!**

## 📦 Технологии

- **Язык:** C#
- **UI:** WPF (XAML)
- **База данных:** SQLite (файл `SportsStore.db`)
- **ORM:** Entity Framework Core
- **Пакеты:** `Microsoft.EntityFrameworkCore.Sqlite`
